Ecg Compressed Sensing with Kronecker Technique & Adaptive Dictionary Learning
================================================================================

This documentation provides an overview of the **Ecg Compressed Sensing with Kronecker Technique & Adaptive Dictionary Learning** project, including the functionality provided by the `CompSensePack` package.

The main focus areas include:

- Utilities for handling matrix operations and saving data to CSV
- Loading and analyzing ECG signals from the MIT-BIH Arrhythmia Database
- Signal-to-Noise Ratio (SNR) calculation and plotting signals
- Measurement matrix generation
